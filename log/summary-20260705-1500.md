# 2026-07-05 15:00 — 特殊符号与量词全面测试

## 修改的问题

上一版完成了 `matcher.c` 中 DOT 标签的语义化修复。本次对所有特殊符号和量词进行全面测试。

| 符号 | 匹配字符 | 结果 |

### 字符集合（8/8 通过）
| 模式 | 输入 | 结果 |
|------|------|------|
| `[abc]` | `b` | ✅ |
| `[^abc]` | `d` | ✅ |

| `[A-Z0-9]` | `5` | ✅ |
| `[.]` | `.` | ✅ |
| `[[]` | `[` | ✅ |

### 花括号量词（8/8 通过）
| 模式 | 输入 | 结果 |
|------|------|------|
| `ab{2}` | `abb` | ✅ |
| `ab{2}` | `ab` | ✅（拒绝正确） |
| `ab{2,4}` | `abb` | ✅ |

| `ab{3,}` | `abbbb` | ✅ |
| `ab{3,}` | `abb` | ✅（拒绝正确） |

### 分组（7/7 通过）
| 模式 | 输入 | 结果 |
|------|------|------|
| `(ab)` | `ab` | ✅ |
| `(ab)+` | `abab` | ✅ |
| `(a|b)+` | `abba` | ✅ |
| `(ab){2}` | `abab` | ✅ |
| `a(bc)*` | `abcbc` | ✅ |
| `a(bc)*` | `a` | ✅ |
| `(a(b)c)` | `abc` | ✅（嵌套分组） |

### DOT 语义化标签验证
- `\d+` → DOT 标签 `\d` ✅
- `\w{2,4}` → DOT 标签 `\w` ✅
- `a.b` → DOT 标签 `.` ✅
- `\s+` → DOT 标签 `\s` ✅
- `\D+` → DOT 标签 `\D` ✅
